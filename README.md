JavaScript library example using Emscripten

## Requirements

- POSIX shell
- GNU Make 4.2.1
- Emscripten 3.1.8

## Build

```sh
npm install
npm run prepare
npm run build
```

## Run Demo

```sh
npx serve
```

Then open `demo.html` or `demo-legacy.html` in your browsers and check DevTool
console output.
